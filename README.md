# GuYang17's Blog Generator

The simplest static blog generator in the world.

Write your articles in `./source/articles/`. Filenames must be formatted like `2026-04-19 | Hello World.md`

Run `python3 generator.py` to build the website.

The result will be in `./output/`

You can run `./launch_server.sh` to have a quick look after finishing building.

That's all!
